# Gonvecs - High-End Tech & Streetwear Store

A modern e-commerce platform built with Next.js, Tailwind CSS, and Supabase.

## 🎯 Features

- **User Authentication**: Email-based registration and login
- **Wallet System**: UPI-based top-up with UTR verification
- **Product Catalog**: Browse and filter by categories
- **Shopping Cart**: Add products to cart with persistent storage
- **Wishlist**: Save favorite products
- **Admin Dashboard**: Manage products, categories, orders, and transactions
- **Responsive Design**: Mobile-first approach with dark theme
- **Animations**: Smooth Framer Motion animations throughout
